Doo


O que é?

Um sistema de lista de afazeres que proporciona uma melhor organização das atividades que precisam ser realizadas


Como funciona?

No Doo, você pode criar seções de tarefas, onde nelas você poderá criar cards de listas de tarefas contendo quais tarefas devem ser realizadas


Exemplo escrito:

Poderia-se criar uma seção chamada "Trabalho", onde dentro dela você poderia criar cards de listas de tarefas como "Reunião", "Relatório". Em de cada card, você poderia criar tarefas como "Marcar reunião com o cliente", "Fazer relatório de vendas", "Enviar e-mail para o chefe", etc.
